# World Atlas Pack

This is the placeholder for a future fully tiled offline world atlas.

Current status:

- The running app uses a built-in zoomable atlas rendered from local HTML, CSS, SVG, and JavaScript.
- `content/maps/world-places.json` documents the starter place data format.
- Large raster or vector tiles are intentionally not committed yet.

Before adding tiles, confirm the source license allows redistribution and include attribution in this folder.
