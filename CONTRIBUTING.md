# Contributing to Family Kids Portal

Thanks for helping build a better offline-first place for kids and families.

## Project principles

- Keep kids safe by default.
- Avoid dark patterns, infinite feeds, ads, tracking, and manipulative rewards.
- Prefer offline-first features.
- Keep parent-visible controls clear.
- Do not add medical, psychological, or diagnostic claims.
- Keep activity insights descriptive and local, not judgmental.

## Local development

Open `index.html` directly in a browser.

Optional checks:

```text
npm run check
```

## Contribution areas

- Kid-safe games and brain-training activities.
- Offline wiki packs.
- Offline maps and geography packs.
- Avatar items and wardrobe features.
- Accessibility and reduced-motion improvements.
- Parent dashboard and family controls.
- Documentation and translations.

## Code style

- Plain HTML, CSS, and JavaScript for now.
- No external runtime dependencies unless there is a strong reason.
- Keep saved data local-first.
- Keep features understandable for parents and children.

## Pull requests

Please include:

- What changed.
- Why it helps kids or parents.
- How it was tested.
- Any safety/privacy concerns.
